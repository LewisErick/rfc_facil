Rfc Fácil

Libreria para calcular el Registro Federal de Contribuyentes en México (RFC) en Ruby.

Uso

gem 'rfc_facil'

Calcular el rfc de una persona física es muy sencillo:

rfc = Rfc.new(name: 'Adrián Marcelo', first_last_name: 'Rangel', second_last_name: 'Araujo', day: 27, month: 11, year: 1992)
rfc.to_s # => 'RAAA921127RI6'

Fuente

Esta librería se basa en documentación oficial obtenida por medio del IFAI (Instituto Federal de Acceso a la Información). El documento puede ser consultado en el sitio de INFOMEX con el folio 0610100135506.

Agradecimiento

El código es una version en Ruby de rfc-facil hecha en Java por josketres
